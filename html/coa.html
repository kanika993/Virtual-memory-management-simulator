FIFO (First In First Out)

Definition: This algorithm replaces the oldest page in memory when a new page needs to be loaded. It operates like a queue, where the first page that entered is the first to be removed.

Working: Pages are loaded in a queue; when a page needs to be replaced, the one at the front is removed.

Optimal Page Replacement

Definition: This algorithm replaces the page that will not be used for the longest time in the future. It provides the lowest possible page fault rate but requires knowledge of future page requests, making it impractical for real-world use.

Working: Predicts future requests and replaces the page not needed soonest.

LRU (Least Recently Used)

Definition: This method replaces the page that has not been used for the longest time. It assumes that pages used recently will likely be needed again soon.

Working: Tracks recent page usage and replaces the least recently used page.
